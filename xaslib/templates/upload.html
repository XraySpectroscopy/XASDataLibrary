Upload XDI Spectra
Uploaded files must be an ASCII file no larger than 16Mb with data arranged a table with columns for indivdual arrays, each row representing a different energy point. Only the first 16 columns will be accessible, and these must include columns for Energy or Monochramator Angle, and arrays such as I0, ITRANS, MUTRANS, IFLUOR, MUFLUOR to idenitify the MU data. You will be asked to select columns and fill in values for some metadata including:
  • Sample Name
  • Absorbing Element and Edge
  • Monochromator d-spacing or nominal reflecting cut
  • Energy units
  • Beamline
  • Measurement Mode

For example files, see example data files at Example XDI data and Example XDI data and Initial X-ray Absorption Data.

For a detailed description of XDI files, see XAS Data Interchange Specification.
